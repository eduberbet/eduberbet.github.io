## [010]
Título: LLM (Large Language Models)
Corpo: Motores neurais treinados com bilhões de textos para entender e gerar linguagem humana de forma natural. É a base da "conversa" inteligente da LUMI Core.

## [011]
Título: IA (Inteligência Artificial)
Corpo: A ciência de criar sistemas capazes de realizar tarefas que normalmente exigiriam inteligência humana, como aprender, raciocinar e resolver problemas.

## [012]
Título: IA Socrática
Corpo: Um método pedagógico onde a IA faz perguntas reflexivas em vez de dar respostas prontas, guiando o usuário a descobrir o conhecimento por conta própria.

## [013]
Título: IA Generativa
Corpo: O ramo da IA focado em criar novos conteúdos (textos, imagens, áudios) a partir de padrões aprendidos. É o que permite a criação de respostas dinâmicas.

## [014]
Título: Clean Code (Código Limpo)
Corpo: A filosofia de escrever códigos que humanos consigam ler e entender facilmente. Código elegante é código simples, sustentável e eficiente.

## [015]
Título: Refatoração
Corpo: O processo de reestruturar um código existente sem alterar seu comportamento externo, visando melhorar a legibilidade e reduzir a complexidade técnica.

## [016]
Título: UI Design (Interface)
Corpo: O foco na estética e na interação visual. Garante que os botões e cores da Rocket sejam funcionais e visualmente atraentes.

## [017]
Título: UX Design (Experiência)
Corpo: O estudo da jornada do usuário. O objetivo é garantir que a interação com o software seja satisfatória, produtiva e sem atritos.

## [018]
Título: PWA (Progressive Web App)
Corpo: Um aplicativo que roda no navegador, mas oferece experiência de app nativo, podendo funcionar offline e ser instalado na tela inicial.

## [019]
Título: HTML
Corpo: A linguagem de marcação que define a estrutura fundamental de uma página web, organizando o conteúdo em blocos lógicos.

## [020]
Título: CSS
Corpo: A linguagem de estilos usada para definir o visual de uma página HTML, incluindo cores, fontes, layouts e animações fluidas.

## [021]
Título: JS (JavaScript)
Corpo: A linguagem de programação que traz interatividade para a web. É o "músculo" que processa a lógica e as ações em tempo real.
